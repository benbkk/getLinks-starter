# An Experiment of Bootstrap Implementation with React (That I hate the least)
Definitely **not** a recommendation or an example how to implement Bootstrap in your react project.

## Getting Started

```
git clone && cd
yarn
```

### Development
```
yarn dev
```

### Production
```
yarn build
```
